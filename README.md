# Sleep Disorder Prediction — ML API with FastAPI & Docker

Predicts **Sleep Disorder** (`None` / `Insomnia` / `Sleep Apnea`) from lifestyle
and health metrics using a **Random Forest** classifier trained on the
[Sleep Health and Daily Performance Dataset](https://www.kaggle.com/datasets/mohankrishnathalla/sleep-health-and-daily-performance-dataset/data).

---

## Project Structure

```
ml-fastapi-docker/
├── train.py          # Train & save the RandomForest model
├── main.py           # FastAPI application (GET / and POST /predict)
├── model.joblib      # Saved model + encoders (generated by train.py)
├── requirements.txt  # Python dependencies
├── Dockerfile        # Container instructions
└── README.md
```

---

## Quick Start (local)

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the model
```bash
python train.py
# → model.joblib is created
```

### 3. Run the API
```bash
uvicorn main:app --reload --port 8000
```

### 4. Test endpoints

| Method | URL | Description |
|--------|-----|-------------|
| GET | `http://localhost:8000/` | Health check |
| POST | `http://localhost:8000/predict` | Sleep disorder prediction |
| GET | `http://localhost:8000/docs` | Swagger UI |
| GET | `http://localhost:8000/redoc` | ReDoc UI |

#### Example POST /predict
```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "gender": "Male",
    "age": 35,
    "occupation": "Software Engineer",
    "sleep_duration": 6.0,
    "quality_of_sleep": 4,
    "physical_activity_level": 40,
    "stress_level": 8,
    "bmi_category": "Overweight",
    "heart_rate": 80,
    "daily_steps": 4000,
    "blood_pressure_systolic": 130,
    "blood_pressure_diastolic": 85
  }'
```

Expected response:
```json
{
  "prediction": "Insomnia",
  "probabilities": {
    "Insomnia": 0.87,
    "None": 0.10,
    "Sleep Apnea": 0.03
  }
}
```

## Docker

### Build the image
```bash
docker build -t sleep-disorder-api .
```
The Dockerfile runs `train.py` during the build, so `model.joblib` is
baked into the image.

### Run the container
```bash
docker run -d -p 8000:8000 --name sleep-api sleep-disorder-api
```

### Test from inside the container (optional)
```bash
docker exec -it sleep-api \
  curl -s http://localhost:8000/
```

### Stop & remove
```bash
docker stop sleep-api && docker rm sleep-api
```

## Model Details

| Property | Value |
|----------|-------|
| Algorithm | Random Forest Classifier |
| Features | 12 (gender, age, occupation, sleep duration, quality of sleep, physical activity, stress level, BMI, heart rate, daily steps, systolic BP, diastolic BP) |
| Target classes | None · Insomnia · Sleep Apnea |
| Train / Test split | 80 / 20 |
